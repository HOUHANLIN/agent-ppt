LAYOUT 01核心观点 + 解释要点

核心观点页：一句大判断 + 3–4 个支撑要点

最适合放“本页想让观众记住的一句话”，例如研究问题、诊断结论、章节总论点。
一句话主张 / Take-home Message
  • 先给结论，降低听众理解成本
  • 每个要点只保留一个动作或判断
  • 适合汇报开头、章节过渡、总结页

右侧可放：证据摘要

用 2–3 句话解释为什么这个观点成立；也可以替换成图片、病例摘要或研究假设。

版式注释：适合 开场观点 / 关键结论 / 章节高潮 ｜ 使用建议：当信息不复杂但需要“先立住观点”时使用。主句要短，辅助信息不超过 3 条。

LAYOUT 02图文解释

图文解释页：左侧讲图，右侧提炼读图结论

图片 / 图表占位
建议尺寸：约 480 × 360
保留图注，避免图片塞满导致无法读图
读图顺序：先看分组 → 再看趋势 → 最后说结论。
  • 看对象：图中比较了哪些组？
  • 看变化：指标升高、降低，还是无差异？
  • 看意义：这个变化支持了什么判断？
版式注释：适合 实验图 / 病理图 / 机制图解读 ｜ 使用建议：适合 Figure 解读、影像学表现、病理形态、网页或软件截图说明。

LAYOUT 03四卡片并列

四卡片并列页：把同一层级的信息横向展开

最适合“并列关系”，例如四个临床表现、四类证据、四个研究步骤；不适合放因果链。

① 背景

说明问题从哪里来，为什么值得讨论。

② 现象

列出观察到的关键表现或数据变化。

③ 机制

解释背后可能的原因、通路或逻辑。

④ 意义

落到诊断、治疗、启发或后续研究。

版式注释：适合 分类 / 四个原因 / 四项证据 ｜ 使用建议：卡片标题尽量同构，例如都用名词或都用动宾短语。

LAYOUT 19Q&A 预案

问答页：提前准备高概率追问

Q1：概念追问

这个概念和另一个概念有什么区别？

A：先定义，再说关键差异，最后回到本汇报场景。

Q2：证据追问

你的判断依据是什么？

A：按“现象—数据—结论”三步回答。

Q3：局限追问

这个方案有什么不足？

A：承认边界，再说明仍然有价值的部分。

Q4：应用追问

这个结论能用在哪里？

A：给一个具体场景，不要泛泛而谈。

版式注释：适合 答辩问题 / 老师追问 / 观众互动 ｜ 使用建议：适合答辩备用页。正式汇报可隐藏，提问时跳转使用。
